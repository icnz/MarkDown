Markdown是一种极简的『标记语言』，其不追求大而全，简洁至上，能方便地将文本转为HTML。

### 标题

##### >> 语法

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

##### >> 示例

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

### 超链接

##### 语法

```markdown
》文字链接 
[链接名称](链接url)
[链接名称](链接url "链接提示信息")

》网址链接 
<url地址>

》变量链接
用 yahoo 作为网址变量 [Yahoo][yahoo]
然后在文档的结尾为变量赋值（网址）
[yahoo]: http://www.yahoo.com "提示信息"
```

##### 示例

》文字链接
[百度](http://baidu.com)
[百度](http://baidu.com "百度搜索")

》网址链接
<http://baidu.com>

》变量链接
用 yahoo 作为网址变量 [Yahoo][yahoo]
然后在文档的结尾为变量赋值（网址）

[yahoo]: http://www.yahoo.com

### 转义字符

##### >> 语法

```markdown
Markdown中的转义字符为\，转义的有：
\\ 反斜杠
\` 反引号
\* 星号
\_ 下划线
\{\} 大括号
\[\] 中括号
\(\) 小括号
\# 井号
\+ 加号
\- 减号
\. 英文句号
\! 感叹号
```

##### >> 示例
